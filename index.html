About

I'm Savindu Damsara — a 16-year-old coder from Sri Lanka who started programming out of pure curiosity and hasn't stopped since.

I spend most of my time in Python, C++, and Bash. Right now I'm going deep on C++ and messing around with Termux because having a dev environment in your pocket is just too good.

I'm interested in cybersecurity — not as a badge, but because understanding how things break is the most honest way to understand how things work. Still figuring it all out, but building every day.

Interests

Coding
Writing code is the hobby. Python scripts, C++ experiments, whatever problem is interesting that day — I just sit down and build.
Bash Scripting
Automating boring things, building tools in the terminal, Termux on Android. There's something satisfying about making a shell script that just works.
Building Hacking Tools
Creating scripts and tools that poke at how systems work. Encrypters, scanners, automation — built to learn, not to harm. Security through curiosity.
Discord Server Creation
Building communities from scratch — setting up bots, permissions, channels, vibes. I like the design problem of making a server feel alive.
Learning How Things Break
Reading about CVEs, messing with Termux tools, watching hacking demos. Not for the flex — just because understanding vulnerabilities is the best way to understand systems.
Making Random Stuff
Games, bots, mini-projects, experiments. If an idea sounds fun, I just start building it. Half of them don't finish. That's fine — the process is the point.
